# 智能家居系统(安卓端)

![image](https://github.com/a-voyager/SmartHouse_Android/raw/master/screenshots/icon.png "图标")

# 介绍
安卓端通过HTTP心跳连接实时获取最新数据，并展示于主界面；同时用户可通过点击按钮控制家中的电器设备，如窗帘、警报等
